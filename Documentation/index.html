Functions:

  • eztxt.write(name, data) - Writes, where data is 'data' and the name of the file is 'name'.
  • eztxt.read(name) - Reads, where output is returned as 'data' and the name of the file is 'name'.
  • eztxt.readline(name, line) - Reads the specified 'line' in the document 'name'.
  • eztxt.readlines(name) - Reads all lines in the file 'name' and returns them seperately in a list.
  • eztxt.append(name, data) - Appends, where data is 'data' and the name of the file is 'name'.
  • eztxt.appends(name, data) - Appends, where data is 'data' and the name of the file is 'name', and adds a space before the start of the data.
  • eztxt.appendn(name, data) - Appends, where data is 'data' and the name of the file is 'name', and puts new data on a new line.

Functions:

  • ezcrypt.encrypt(data, key) - Encrypts 'data' with the key 'key' using AES-256 bit encryption. Returns the encrypted string.
  • ezcrypt.decrypt(data, key) - Decrypts 'data' with the key 'key' using AES-256 bit encryption. Returns the decrypted string.
‎

Functions:

  • ezpassword.check(input_, correcthash) - Compares the user's inputted password 'input_' against the known hash of the correct password 'correcthash'. If they are matching, returns True, and if not, False.
  • ezpassword.newpass(input_) - Returns the sha256 hash of 'input_'.
  • ezpassword.generate(sha256output, length) - Generates a alphanumeric password of 'length' length, the default being 7 characters. 'sha256output' lets you choose whether the string returned will be sha256 or not. If 'sha256output' is 0 (default), it will return the generated password. If 'sha256output' is 1, it will return the sha256 hash of the generated password. If 'sha256output' is 2, it will return the sha256 hash and the password generated in a list in the format [password, sha256].

Functions:

  • ezlog.log(data) - Saves a log with the date and time and then 'data' in a folder called 'logs' under the filename of '[today's date].log'. Also prints the log. MUST be in the same folder as eztxt.
